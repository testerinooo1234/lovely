# lovely

**Live site:** https://testerinooo1234.github.io/lovely/

A dark, taboo erotica reading site — plain dirty stories, frontend only.

## Stack

- Vite + React + TypeScript
- React Router

Stories and tags live in `src/data/stories.ts`.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
